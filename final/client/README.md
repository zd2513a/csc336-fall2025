This full-stack Recipe Manager application allows users to add, view, and explore recipe cards with a clean interactive UI.

Features
	•	Add new recipes through a responsive form
	•	Flip-card UI to reveal detailed ingredients and steps
	•	Persistent storage using data.json on the backend
	•	Real-time updates (new recipes appear instantly)
	•	Clean navigation across Home, Recipe List, and Add Recipe
	•	Fully deployed and publicly accessible

External library
    •   react-toastify
